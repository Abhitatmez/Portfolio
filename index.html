Welcome to My macOS Portfolio!

This isn't your typical portfolio website - it's an interactive macOS-inspired experience. I wanted to create something unique that showcases both my work and my creativity.

Feel free to explore and interact with the interface just like you would on a real Mac!

How to Navigate

Click on any app icon to open its window. You can:

  • Move windows by dragging their top bar
  • Resize windows using the maximize button
  • Close windows using the red button
  • Minimize windows to the dock

Customization

Make it yours:

  • Toggle between light and dark mode using the theme selector
  • Drag windows anywhere on the screen
  • Arrange icons as you prefer

The Menubar

Just like macOS, the menubar contains dynamic elements:

  • Real-time date and time display
  • Live battery status and percentage
  • WiFi status
  • Theme toggle (dark/light)
  • Drag icons toggle (enabled/disabled)

Available Apps

Click these icons to explore:

  • About Me - Learn about who I am
  • Resume - View my professional experience
  • Certificates - Check my certifications
  • Projects - See what I've built
  • Contact - Check the menubar for contact options
